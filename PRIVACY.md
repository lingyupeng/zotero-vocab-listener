# Privacy

Zotero Vocab Listener is local-first.

The plugin reads Zotero annotation metadata from your local Zotero profile and
stores vocabulary records in Zotero preferences. It does not send collected
words, translations, paper titles, PDF content, or annotation metadata to any
external server.

Exports are written only to the local file path you choose.

No analytics, telemetry, tracking, or network calls are included in the plugin
runtime.
